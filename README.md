# Russian Age Tkinter

Программа на Python с графическим интерфейсом для демонстрации правильных склонений слова "год" в русском языке.

## Функциональность

- Генерация фраз от 1 до 99 лет с правильными окончаниями
- Графический интерфейс на tkinter
- Автоматическая прокрутка списка результатов
- Адаптация под русские правила склонения

## Интерфейс

- Кнопка "Возраст" — запускает генерацию
- Текстовое поле с прокруткой — вывод всех фраз
- Аккуратный дизайн через ttk

## Логика склонения

def get_age_string(age):
    if age % 10 == 1 and age % 100 != 11:
        return f"Мне {age} год"
    elif 2 <= age % 10 <= 4 and (age % 100 < 10 or age % 100 >= 20):
        return f"Мне {age} года"
    else:
        return f"Мне {age} лет"

## Примечание

Программа демонстрирует правильную обработку исключений русского языка (числа 11-14 — особый случай). Кнопка запускает цикл от 1 до 99, применяя функцию склонения к каждому числу.
